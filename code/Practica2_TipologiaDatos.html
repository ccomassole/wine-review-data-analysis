PRACTICA2. Tipologia de Datos¶

Preparación y analisis de un dataset¶

Script de preparacin y anlisis del Dataset "Wine Review". El objetivo planteado es verificar si existen relaciones significativas entre entre el precio de un vino, su valoracin y la descripcin que del mismo realiza un sommelier experto. Para simplificar el anlisis, se tomar como elemento de valoracin de la descripcin la longitud de la misma. De este modo el analisis se centrar exclusivamente en valores numericos.

La numeracin dada a cada apartado se corresponde con la del documento explicativo.

# Contamos los registros vacios y calculamos el porcentaje que representan
def vacios(df):
    muestras_vacias = df.isnull().sum()
    # identificamos su peso en cada caso:
    total_registros = np.product(df.shape)
    total_vacias = muestras_vacias.sum()
    return ((float(total_vacias)/float(total_registros)) * 100)
